# 카카오 인가 화면 리다이렉트

## 카카오 인가 화면 리다이렉트

### 개요
카카오 인가 화면으로 리다이렉트합니다. 클라이언트가 카카오 로그인을 시작할 때 호출하는 엔드포인트입니다.

### 엔드포인트
`GET /login/kakao`

### 인증
- **인증 필요 여부:** 불필요
- **권한:** 없음 (비인증 사용자 접근 가능)

### 요청 (Request)

파라미터 없음.

---

### 응답 (Response)

**성공**

| HTTP Status | 의미 |
|-------------|------|
| 302 | Found - 카카오 인가 URL로 리다이렉트 |

응답 본문 없이, `Location` 헤더를 통해 카카오 인가 화면으로 리다이렉트됩니다.
